一、滚轮

只要在css文件夹中写入此段代码，页面的滚动条样式就会改变，但是只限于谷歌浏览器

/*滚轮，只对谷歌浏览器有效*/ /*设置横向滚动条的高度，纵向滚动条的宽度*/ *::-webkit-scrollbar{ width:6px; height:8px; } /*横向滚动条*/ *::-webkit-scrollbar-thumb:horizontal{ /*滚动条圆弧度*/ border-radius:6px; /*滚动条背景颜色*/ background-color:#e1e5ec; } /*纵向滚动条*/ *::-webkit-scrollbar-thumb:vertical{ /*滚动条圆弧度*/ border-radius:6px; /*滚动条背景颜色*/ background-color:#e1e5ec; }
二、分页
1、引入jquery.js文件
2、引入displayPage-min.js
3、在body中写分页要显示的位置,如: <div class="pc-page-num"></div> 4、在页面上写入此段代码： $(".pc-page-num").sjAjaxPager({ url:'接口地址', tbody:$(''),//数据显示的位置 success:function(data){ //规定页面的高度，当一个页面中有多个iframe时 GetIframeBodyHeight(data, 192, 'iframe1');//第一个参数：需要加载的数据、二：页面的没有数据时的高度、三：iframe父页面的的名字 } });
